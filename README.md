# ven-cars

An car owner API

## Scripts

Install dependencies with:
```
npm i
```

Start the development server with the:
```
npm run start:dev
```
Run tests with:
```
npm test
```
Run migrations and seed the db with:
```
npm run migrate
npm run seed
```
You rollback database changes with:
```
npm run rollback
```
